# دليل تصدير تطبيق مركز نور الماسي (APK / AAB)

لتحويل هذا الموقع إلى تطبيق أندرويد حقيقي ونشره في المتجر، اتبع الخطوات التالية على جهاز الكمبيوتر الخاص بك:

## المتطلبات الأساسية:
1.  تثبيت **Node.js** على جهازك.
2.  تثبيت **Android Studio**.
3.  تحميل كود المشروع الحالي.

## الخطوات البرمجية:

### 1. تثبيت المكتبات (في حال لم تكن مثبتة):
افتح "Terminal" في مجلد المشروع ونفذ:
```bash
npm install
```

### 2. بناء نسخة الويب:
```bash
npm run build
```

### 3. إضافة منصة الأندرويد للمشروع:
```bash
npm run cap:add
```

### 4. مزامنة الكود مع الأندرويد:
```bash
npm run cap:sync
```

### 5. فتح المشروع في Android Studio:
```bash
npm run cap:open
```

---

## داخل Android Studio (لتوليد الملف النهائي):

### للحصول على ملف APK (للتثبيت المباشر):
1.  من القائمة العلوية اختر: **Build** > **Build Bundle(s) / APK(s)** > **Build APK(s)**.
2.  انتظر حتى ينتهي، ثم اضغط على **locate** في الإشعار الذي سيظهر لك لتجد ملف الـ APK.

### للحصول على ملف AAB (للنشر في متجر جوجل بلاي):
1.  من القائمة العلوية اختر: **Build** > **Generate Signed Bundle / APK...**.
2.  اختر **Android App Bundle** ثم اضغط **Next**.
3.  قم بإنشاء مفتاح توقيع جديد (Key Store) إذا لم يكن لديك واحد، واتبع الخطوات.
4.  في النهاية ستحصل على ملف بصيغة `.aab` جاهز للرفع على "Google Play Console".

---

**ملاحظة:** لقد قمت مسبقاً بضبط إعدادات التطبيق (App ID: `com.nooralmasi.app`) واسم التطبيق في ملف `capacitor.config.json`.
